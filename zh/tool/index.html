# 工具

# create-strve

在前面我们也简单介绍过，create-strve是基于Strve.js的项目构建工具，您可以使用它更方便灵活地搭建页面。create-strve是用Vite (opens new window)来构建的，它是一种新型前端构建工具，能够显著提升前端开发体验。

# 安装

全局安装

npm install create-strve -g

查看版本

create-strve -v

初始化项目

create-strve init <projectName>

# 启动

yarn dev
# OR
npm run dev

# 部署

yarn build
# OR
npm run build

# 配置

因为create-strve是用Vite (opens new window)来构建的，所以你可以按照Vite (opens new window)的约定配置进行自定义配置create-strve。
